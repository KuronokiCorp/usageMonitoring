# Worklogs — usageMonitoring / itermon (company rule 7)

Every agent keeps a daily action record here: `docs/worklog/<agent-slug>/YYYY-MM-DD.md`.
Write an entry at the end of every working session; read your own last 5 at session start.
That accumulated record is the agent's memory and personality.

feature/develop branch. Monthly rollups: `<agent>/rollups/YYYY-MM.md`.
